# If you need to cancel or rearrange your appointment:

* visit https://www.nhs.uk/book-vaccine
* if you can't get online, call 119

Please cancel your appointment if you have had a flu vaccine somewhere else. It helps the pharmacy to let them know if you can no longer attend.

You can cancel your appointment up until the day it takes place. However, we cannot change appointments an hour before they are due to start.
